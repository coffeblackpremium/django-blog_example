Comandos básicos para Linux

Com um aumento estrondoso de Usuarios, o linux vem cada dia a mais conquistando mais espaço para pessoas que estão se aprofundando em uma nova aventura, seja para Trabalho e estudo, ou até mesmo, para Lazer e Pessoal.

Atualmente, existem diversas distribuições Linux, cada uma com sua particularidade e diferenças, uma vantagem aqui outra vantagem ali, mas em Geral, há uma coisa que é sempre estática: Os comandos Terminais Linux (Lógico, isto é, excluindo os comandos de Gerenciadores de Pacotes que isto seria assunto para outro Post).

Nesta postagem, irei listar para vocês alguns comandos básicos para você manipular no seu dia-a-dia. Seja como eu disse, para Trabalho, Estudos e também para Lazer.

Whoami

PWD

O PWD digitado no Terminal, ele ira mostrar em que pasta você está no Terminal.

É um comando muito utilizado e muito util para o nosso dia-a-dia. Ajudando a gente caso se perdessemos no "terminal"

Change Directory (CD)

Se quisermos ir para uma determinada pasta, bastamos utilizar o comando CD + nome da pasta
